# Cognifyz Data Analysis Internship

This repository contains my work for the Data Analysis Internship at Cognifyz Technologies.
The project focuses on data cleaning, exploratory data analysis, visualization, and deriving
meaningful insights from a real-world restaurant dataset.

---

## Level 1 Tasks Completed

- Task 1: Top Cuisines Analysis
- Task 2: City-wise Restaurant Analysis
- Task 3: Price Range Distribution
- Task 4: Online Delivery Analysis
- Data Cleaning and Preprocessing
- Data Visualization using Python

---

## Tools & Technologies

- Python (Pandas, NumPy, Matplotlib)
- Google Colab
- GitHub

---

## Key Highlights

- Cleaned and preprocessed a real-world restaurant dataset
- Identified top cuisines and their distribution across restaurants
- Performed city-wise analysis based on restaurant count and ratings
- Analyzed price range distribution using donut and percentage charts
- Compared average ratings based on online delivery availability
- Used appropriate visualization techniques for different analysis tasks
